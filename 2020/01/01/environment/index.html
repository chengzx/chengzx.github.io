开发环境配置

Jan 1, 2020 | 开发

Java 开发环境配置

  • JAVA_HOME jdk安装/解压目录
  • Path %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin
  • CLASSPATH %JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar
  • 验证 win+R cmd java -version

MySql 免安装版本

  • 解压缩 D:/mysql
  • 复制 my-default.ini –> my.ini 或者新建 my.ini
  • 复制下面的内容到my.ini文件 （自行创建data文件夹）
  • 添加环境变量
    • 配置 MYSQL_HOME D:\Program Files Dev (x86)\mysql-5.7.17-winx64
    • Path 追加 %MYSQL_HOME%\bin
  • 将 MySql 注册为 Windows 系统服务
    • 以管理员的身份从控制台进入到MySQL解压目录下的 bin 目录下 cd D:/mysql/bin
    • 执行命令初始化MySql mysqld --initialize-insecure --user=mysql
    • 启动 MySql
    • 添加系统服务
  • 启动MySQL服务 net start mysql
  • 关闭MySQL服务 net stop mysql
  • 移除MySQL服务 mysqld --remove mysql

Maven

  • M2_HOME D:\apache-maven-3.5.0
  • Path %M2_HOME%\bin;
  • mvn -version
