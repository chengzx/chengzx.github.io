安装Tomcat

  1. 必须先安装jdk
  2. tar 解压缩
  3. conf 路径下的 server.xml 中8080端口后面加字符集配置 URIEncoding=‘UTF-8’
  4. bin 路径 startup.sh 执行
  5. 浏览器访问 centosIP:8080 显示tomcat管理页面即为成功
  6. 如果 Linux 启动 Tomcat 成功，但是浏览器不能打开 Tomcat 管理页面原因是 Linux 的8088端口没有打开
  7. Centos7 以下版本

Nexus

  1. 解压缩Nexus /usr/devleop/nexus
  2. 进入解压路径 cd nexus/nexus**
  3. 配置文件 ./etc 下启动文件 ./bin
  4. 修改端口 vi ./etc/nexus-deafault.properties application-port application-host
  5. 启动/重启/状态/停止 Nexus 服务
  6. 开机自动启动

MySql

  1. 创建用户组与用户
  2. 解压缩 tar -zxvf mysql-5.7.19XXXXXXX -C 目标目录（mysql默认安装目录/usr/local/mysql）
  3. 初始化
  4. 启动
  5. 修改密码与链接权限
  6. 关闭防火墙
  7. 添加用户赋权限
  8. 找回密码

Nginx

  1. 前置安装准备 yum -y install gcc zlib zlib-devel prce-devel openssl openssl-devel
  2. tar解压缩 Nginx
  3. 编译
  4. 验证 cd安装路径/sbin/nginx -t
  5. 常用命令
  6. 注意80端口
  7. 域名配置与测试验证
  8. 注意事项 域名转发,一定要配置host并使之生效，重启浏览器
